MEUS PROJETOS.

Projeto 1

O DevCurrency permite acompanhar em tempo real as taxas de câmbio de criptomoedas, oferecendo uma interface intuitiva para visualizar variações de preços. Desenvolvido em React.js com TypeScript e integração via API.

Projeto 2

A Barber Shop é é uma landing page de uma barbearia, com foco em um design moderno e atraente para destacar os serviços oferecidos. Foi desenvolvido utilizando HTML, SCSS para estilização avançada e JavaScript para interatividade, proporcionando uma experiência visual limpa e funcional.

Projeto 3

Um quiz interativo com perguntas sobre programação, desenvolvido para testar os conhecimentos dos usuários. Utilizando HTML para a estrutura, CSS para o layout e JavaScript para controlar a lógica das respostas e a navegação entre as questões.
